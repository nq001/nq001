## Currently Building

### API Monitor Enterprise

A production-oriented API Management and Performance Monitoring Platform built to help organizations understand how their APIs behave in production.

Instead of only testing APIs, the platform continuously observes live traffic, measures latency, detects failures, performs request logging, applies rate limiting, and provides operational insights.

Core capabilities include:

- JWT Authentication
- Role-Based Access Control
- API Registration
- Endpoint Management
- Request Interception
- Redis Buffering
- Batch PostgreSQL Persistence
- Analytics APIs
- Alert Evaluation
- Docker Deployment

---

### FundChain

A blockchain-powered crowdfunding platform designed to provide transparent and secure fundraising through Ethereum Smart Contracts.

The platform automates milestone-based fund releases, contributor refunds, reward distribution, and campaign verification while removing reliance on centralized intermediaries.

Core capabilities include:

- Smart Contracts
- MetaMask Integration
- Milestone Escrow
- KYC Verification
- Blockchain Event Tracking
- Multi-role Dashboards
